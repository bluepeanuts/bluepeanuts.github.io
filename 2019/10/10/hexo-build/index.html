可能遇到的问题及解决方法

  1. npm install出错

    sudo npm install

  2. 添加gitalk后所有页面都是评论

    手动设置comments: false

  3. hexo d时出错

    在修改.yml文件时，冒号后面一定要有空格

  4. 关于页404

    需要新建关于页，见上方的参考资料

  5. 删除github里博客的commit记录

    cd myblog -> ls -la -> rm -rf .deploy_git
